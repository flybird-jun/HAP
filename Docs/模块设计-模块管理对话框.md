# 模块管理对话框模块设计文档

## 一、模块概述

### 1.1 模块名称
`module_manager_dialog.py`

### 1.2 所属层级
UI层

### 1.3 职责描述
提供模块管理界面，支持模块的添加、编辑、删除操作。

```

|                                                                            |

|  │  ┌─────┬──────────────────────────┬──────────┬───────────────────┐   │  |
```

### 2.2 界面元素说明

| 元素 | 类型 | 说明 |
|------|------|------|
| 模块名称输入框 | QLineEdit | 输入新模块名称 |
| 添加按钮 | QPushButton | 添加新模块 |
| 模块列表表格 | QTableWidget | 显示所有模块 |
| 编辑按钮 | QPushButton (每行) | 编辑模块名称 |
| 删除按钮 | QPushButton (每行) | 删除模块 |
| 统计信息 | QLabel | 显示模块总数和关联问题数 |
| 关闭按钮 | QPushButton | 关闭对话框 |

## 七、操作流程

### 7.1 添加模块流程

```
用户输入模块名称
    │
    v
点击添加按钮
    │
    v
验证名称
    │
    ├─ 无效 → 显示错误 → 返回
    │
    v 有效
调用 ModuleDAO.create()
    │
    ├─ 成功 → 显示成功 → 清空输入 → 刷新列表
    │
    └─ 失败 → 显示错误 → 返回
```

### 7.2 编辑模块流程

```
用户点击编辑按钮
    │
    v
弹出编辑对话框（输入新名称）
    │
    ├─ 取消 → 返回
    │
    v 确认
验证新名称（排除当前模块）
    │
    ├─ 无效 → 显示错误 → 返回
    │
    v 有效
调用 ModuleDAO.update()
    │
    ├─ 成功 → 显示成功 → 刷新列表
    │
    └─ 失败 → 显示错误 → 返回
```

### 7.3 删除模块流程

```
用户点击删除按钮
    │
    v
检查关联问题单数量
    │
    v
弹出确认对话框（显示关联数量）
    │
    ├─ 取消 → 返回
    │
    v 确认
调用 ModuleDAO.delete()
    │
    ├─ 成功 → 显示成功 → 刷新列表
    │
    └─ 失败 → 显示错误 → 返回
```
