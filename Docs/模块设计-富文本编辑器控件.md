# 富文本编辑器控件设计文档

## 一、模块概述

### 1.1 模块名称
`rich_text_editor.py`

### 1.2 所属层级
UI层 - 自定义控件

### 1.3 职责描述
提供富文本编辑功能，支持文字格式化和图片粘贴，用于问题单描述等需要图文混排的场景。

## 四、图片粘贴处理逻辑

### 4.1 支持的粘贴来源

| 来源 | 格式 | 处理方式 |
|------|------|----------|
| 剪贴板截图 | PNG/JPEG | 直接保存 |
| 复制的图片文件 | PNG/JPEG/GIF | 保存文件 |
| 浏览器复制的图片 | Base64 | 解码保存 |
| 截图工具 | PNG | 直接保存 |

### 4.2 粘贴事件处理流程

```
用户 Ctrl+V 粚贴
    │
    v
检测剪贴板内容类型
    │
    ├─ 文本 → 正常粘贴文本
    │
    ├─ 图片 → 提取图片数据 → 保存到本地 → 插入HTML标签
    │
    ├─ HTML（含base64图片） → 提取base64 → 保存 → 替换路径
    │
    └─ 文件列表 → 检查是否图片文件 → 保存 → 插入
```

### 4.3 图片显示规格

| 属性 | 值 | 说明 |
|------|------|------|
| 最大宽度 | 600px | 图片显示最大宽度 |
| 缩放方式 | 自适应 | 保持比例缩放 |
| 对齐方式 | 居中 | 图片居中显示 |

## 八、使用示例

### 8.1 基本使用

```python
from src.ui.widgets.rich_text_editor import RichTextEditor

# 创建编辑器
editor = RichTextEditor(placeholder="请输入详细描述...")

# 设置最小高度
editor.set_min_height(200)

# 获取内容
html_content = editor.get_html_content()

# 设置内容
editor.set_html_content("<p>已保存的内容</p><img src='images/xxx.png'>")

# 监听内容变更
editor.content_changed.connect(lambda html: print("内容已变更"))

# 监听图片插入
editor.image_inserted.connect(lambda path: print(f"图片插入: {path}"))
```

## 九、与ImageManager集成

编辑器需要与 ImageManager 配合使用：

```python
class RichTextEditor(QWidget):
    def __init__(self, issue_no: str = None, ...):
        self.issue_no = issue_no
        self.image_manager = ImageManager()

    def save_pasted_image(self, image_data: bytes) -> str:
        """保存图片并返回相对路径"""
        if self.issue_no:
            return self.image_manager.save_image(image_data, self.issue_no)
        return None
```
