# 问题单创建对话框模块设计文档

## 一、模块概述

### 1.1 模块名称
`create_issue_dialog.py`

### 1.2 所属层级
UI层

### 1.3 职责描述
提供问题单创建界面，用户输入简要描述、详细描述，选择模块（可选），生成问题单号并保存。

## 四、问题单号生成逻辑

```python
def generate_issue_no(self) -> str:
    """
    生成问题单号

    格式: PR + YYYYMMDD + HHMMSS

    示例: PR20260417223510

    Returns:
        问题单号字符串
    """
    now = datetime.now()
    return "PR" + now.strftime("%Y%m%d%H%M%S")
```

## 五、输入验证规则

### 5.1 简要描述验证

| 规则 | 说明 |
|------|------|
| 必填 | 不能为空或纯空格 |
| 长度限制 | 最多100个字符 |
| 格式 | 纯文本，不支持特殊格式 |

### 5.2 详细描述验证

| 规则 | 说明 |
|------|------|
| 必填 | 不能为空或纯空格 |
| 格式 | 富文本，支持图片 |
| 图片处理 | 粘贴的图片自动保存到本地 |

### 5.3 模块选择验证

| 规则 | 说明 |
|------|------|
| 可选 | 可以不选择模块 |
| 默认值 | 无模块关联 |

## 七、提交流程

```
用户点击提交按钮
    │
    v
验证输入字段
    │
    ├─ 无效 → 显示错误提示 → 返回
    │
    v 有效
生成问题单号
    │
    v
处理详细描述中的图片
    │
    v
创建 Issue 数据模型
    │
    v
调用 IssueDAO.create()
    │
    ├─ 成功 → 显示成功提示 → 发送信号 → 关闭对话框
    │
    └─ 失败 → 显示错误提示 → 返回
```

## 八、使用示例

```python
from src.ui.create_issue_dialog import CreateIssueDialog
from PySide6.QtWidgets import QApplication, QMainWindow

app = QApplication()
parent = QMainWindow()

dialog = CreateIssueDialog(parent)

# 监听创建成功信号
dialog.issue_created.connect(lambda issue_id: print(f"创建成功: {issue_id}"))

if dialog.exec() == QDialog.Accepted:
    print("问题单创建成功")
```
