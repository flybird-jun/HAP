## 三、存储规则

### 3.1 目录结构

```
images/
├── PR20260417223510/        # 按问题单号分目录
│   ├── 1713378910_abc123.png
│   ├── 1713378911_def456.jpg
│   └── ...
├── PR20260418093015/
│   └── ...
└── ...
```

### 3.2 文件命名规则

```
{timestamp}_{random_string}.{extension}
```

- timestamp: Unix 时间戳（秒）
- random_string: 6位随机字母数字
- extension: 原始扩展名（小写）

示例：`1713378910_abc123.png`

### 3.3 路径格式

存储在数据库中的相对路径格式：
```
images/PR20260417223510/1713378910_abc123.png
```

## 四、Base64图片提取逻辑

### 4.1 支持的Base64格式

```
data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...
data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...
data:image/gif;base64,R0lGODlhEAAQAMQAAORH...
data:image/webp;base64,UklGRjAAAABXRUJQVlA...
```

### 4.2 提取流程

```
1. 正则匹配 <img src="data:image/...;base64,..."> 标签
2. 提取 MIME 类型和 base64 数据
3. 将 base64 解码为二进制数据
4. 保存到本地文件
5. 替换 HTML 中的 base64 为本地路径
6. 返回处理后的 HTML
```

### 4.3 正则表达式

```python
# 匹配 base64 图片标签
BASE64_IMG_PATTERN = r'<img[^>]+src="data:image/(png|jpeg|jpg|gif|webp);base64,([^"]+)"[^>]*>'

# 匹配 base64 数据 URL
DATA_URL_PATTERN = r'^data:image/(png|jpeg|jpg|gif|webp);base64,(.+)$'
```

## 五、设计要点

### 5.1 目录自动创建
- 初始化时检查 images 目录是否存在
- 保存图片时自动创建问题单子目录

### 5.2 文件去重
- 相同内容的图片不重复保存
- 可选：计算图片哈希值进行去重

### 5.3 错误处理
- 图片保存失败时抛出异常
- 删除不存在的图片时返回 False

### 5.4 路径安全
- 验证路径不包含 `..` 等危险字符
- 确保路径在 images 目录内

## 六、使用示例

```python
from src.data.image_manager import ImageManager

# 初始化
image_manager = ImageManager("/path/to/app")

# 从二进制数据保存图片
with open("screenshot.png", "rb") as f:
    image_data = f.read()
relative_path = image_manager.save_image(image_data, "PR20260417223510")
# 返回: images/PR20260417223510/1713378910_abc123.png

# 从文件保存图片
relative_path = image_manager.save_image_from_file(
    "/tmp/clipboard_image.png",
    "PR20260417223510"
)

# 处理HTML中的base64图片
html = '<p>问题描述：<img src="data:image/png;base64,iVBORw0K..."></p>'
processed_html = image_manager.extract_images_from_html(html, "PR20260417223510")
# 返回: '<p>问题描述：<img src="images/PR20260417223510/1713378910_xyz789.png"></p>'

# 删除图片
image_manager.delete_image(relative_path)

# 删除问题单所有图片
count = image_manager.delete_issue_images("PR20260417223510")

# 检查图片是否存在
if image_manager.image_exists(relative_path):
    print("图片存在")

# 获取图片绝对路径（用于显示）
abs_path = image_manager.get_absolute_path(relative_path)
```
