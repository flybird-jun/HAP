## 三、SQL语句设计

### 3.1 插入模块

```sql
INSERT INTO module (name, created_at)
VALUES (?, ?)
```

### 3.2 查询模块

```sql
SELECT id, name, created_at
FROM module
WHERE id = ?
```

### 3.3 查询所有模块

```sql
SELECT id, name, created_at
FROM module
ORDER BY created_at ASC
```

### 3.4 更新模块

```sql
UPDATE module
SET name = ?
WHERE id = ?
```

### 3.5 删除模块

```sql
DELETE FROM module
WHERE id = ?
```

### 3.6 检查名称是否存在

```sql
SELECT COUNT(*)
FROM module
WHERE name = ? AND id != ?
```

### 3.7 统计关联问题单数

```sql
SELECT COUNT(*)
FROM issue
WHERE module_id = ?
```

### 3.8 获取模块及问题单数量

```sql
SELECT
    m.id, m.name, m.created_at,
    COUNT(i.id) as issue_count
FROM module m
LEFT JOIN issue i ON m.id = i.module_id
GROUP BY m.id
ORDER BY m.created_at ASC
```

## 四、设计要点

### 4.1 名称唯一性
- 模块名称必须唯一
- 创建和更新时检查唯一性
- 数据库层面使用 UNIQUE 约束

### 4.2 关联问题单处理
- 删除模块时，关联的问题单的 module_id 设为 NULL
- 通过外键约束 `ON DELETE SET NULL` 实现
- 删除前可提示用户关联的问题单数量

### 4.3 输入验证
- 名称不能为空
- 名称前后去除空白字符

## 五、使用示例

```python
from src.data.database_manager import DatabaseManager
from src.data.module_dao import ModuleDAO

# 初始化
db = DatabaseManager()
module_dao = ModuleDAO(db)

# 创建模块
module_id = module_dao.create("用户管理")

# 检查名称是否存在
if module_dao.is_name_exists("用户管理"):
    print("模块名已存在")

# 获取所有模块
modules = module_dao.get_all()
for module in modules:
    print(f"{module.id}: {module.name}")

# 更新模块
module_dao.update(module_id, "用户权限管理")

# 删除模块前检查
issue_count = module_dao.count_issues(module_id)
if issue_count > 0:
    print(f"该模块关联了 {issue_count} 个问题单，删除后问题单将取消关联")

# 删除模块
module_dao.delete(module_id)
```
