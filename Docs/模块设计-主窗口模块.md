# 主窗口模块设计文档

## 一、模块概述

### 1.1 模块名称
`main_window.py`

### 1.2 所属层级
UI层

### 1.3 职责描述
系统的主界面，展示问题单列表，提供搜索、过滤功能，以及问题单创建、模块管理的入口。

## 六、停留时间计算逻辑

```python
def calculate_stay_duration(self, status_changed_at: str, status: int) -> str:
    """
    计算并格式化停留时间

    Args:
        status_changed_at: 状态变更时间字符串
        status: 当前状态

    Returns:
        格式化的停留时间字符串，关闭状态返回 "--"
    """
    # 关闭状态不显示停留时间
    if status == IssueStatus.CLOSED:
        return "--"

    # 计算时间差
    change_time = datetime.strptime(status_changed_at, "%Y-%m-%d %H:%M:%S")
    now = datetime.now()
    delta = now - change_time

    # 格式化输出
    days = delta.days
    hours = delta.seconds // 3600
    minutes = (delta.seconds % 3600) // 60

    if days > 0:
        return f"{days}天{hours}小时"
    elif hours > 0:
        return f"{hours}小时{minutes}分钟"
    else:
        return f"{minutes}分钟"
```
