## 三、表结构设计

### 3.1 问题单表 (issue)

```sql
CREATE TABLE IF NOT EXISTS issue (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    issue_no TEXT NOT NULL UNIQUE,           -- 问题单号 PRYYYYMMDDHHMMSS
    title TEXT NOT NULL,                      -- 简要描述
    description TEXT,                          -- 详细描述（支持图片路径）
    module_id INTEGER,                         -- 关联模块ID
    status INTEGER NOT NULL DEFAULT 0,        -- 状态码
    root_cause TEXT,                          -- 问题原因
    solution TEXT,                            -- 问题修改
    self_test TEXT,                           -- 自测试
    archive_test TEXT,                        -- 归档测试描述
    created_at TEXT NOT NULL,                 -- 创建时间
    updated_at TEXT NOT NULL,                 -- 更新时间
    status_changed_at TEXT NOT NULL,          -- 最后状态变更时间
    FOREIGN KEY (module_id) REFERENCES module(id) ON DELETE SET NULL
);
```

### 3.2 模块表 (module)

```sql
CREATE TABLE IF NOT EXISTS module (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,                -- 模块名称
    created_at TEXT NOT NULL                  -- 创建时间
);
```

### 3.3 图片表 (image)

```sql
CREATE TABLE IF NOT EXISTS image (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    issue_id INTEGER NOT NULL,                -- 关联问题单ID
    field_name TEXT NOT NULL,                 -- 所属字段
    file_path TEXT NOT NULL,                  -- 图片存储路径
    created_at TEXT NOT NULL,                  -- 创建时间
    FOREIGN KEY (issue_id) REFERENCES issue(id) ON DELETE CASCADE
);
```

### 3.4 索引设计

```sql
-- 问题单号索引（唯一）
CREATE UNIQUE INDEX IF NOT EXISTS idx_issue_no ON issue(issue_no);

-- 问题单状态索引
CREATE INDEX IF NOT EXISTS idx_issue_status ON issue(status);

-- 问题单创建时间索引
CREATE INDEX IF NOT EXISTS idx_issue_created ON issue(created_at);

-- 模块名称索引
CREATE UNIQUE INDEX IF NOT EXISTS idx_module_name ON module(name);

-- 图片关联索引
CREATE INDEX IF NOT EXISTS idx_image_issue ON image(issue_id);
```

## 四、设计要点

### 4.1 单例模式
使用单例模式确保应用程序全局只有一个数据库连接实例，避免资源浪费和并发问题。

### 4.2 连接管理
- 使用 `check_same_thread=False` 允许多线程访问
- 使用 `isolation_level=None` 手动控制事务
- 启用外键约束 `PRAGMA foreign_keys = ON`

### 4.3 时间存储
所有时间字段使用 TEXT 类型存储 ISO 8601 格式字符串：
```
YYYY-MM-DD HH:MM:SS
```

### 4.4 错误处理
- 捕获 sqlite3.Error 异常
- 记录错误日志
- 提供友好的错误信息

## 五、使用示例

```python
# 获取数据库管理器实例
db = DatabaseManager()

# 初始化表结构
db.init_tables()

# 执行查询
issues = db.fetchall("SELECT * FROM issue WHERE status = ?", (0,))

# 执行插入
db.execute(
    "INSERT INTO issue (issue_no, title, status, created_at, updated_at, status_changed_at) VALUES (?, ?, ?, ?, ?, ?)",
    ("PR20260417223510", "登录失败", 0, datetime.now().strftime("%Y-%m-%d %H:%M:%S"), ...)
)
db.commit()

# 事务操作
try:
    db.begin_transaction()
    db.execute(...)
    db.execute(...)
    db.commit()
except Exception as e:
    db.rollback()
    raise e
```
